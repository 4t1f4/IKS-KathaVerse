# KathaVerse – Oral Forms of Storytelling in India

## Overview

KathaVerse is an interactive educational website created as part of the Indian Knowledge System (IKS) project. The website explores the rich tradition of Oral Forms of Storytelling in India, from ancient folk tales, epics, songs, and performances to contemporary digital storytelling.

The project represents how storytelling has evolved from traditional face-to-face narration to modern platforms like podcasts, social media, and digital experiences.

## Purpose

The goal of this project is to preserve and present India's storytelling traditions in an engaging digital format. It demonstrates how oral storytelling, which was traditionally shared through voices and communities, continues to evolve through modern technology and digital platforms.
