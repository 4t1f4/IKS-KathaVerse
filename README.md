# KathaVerse – Oral Forms of Storytelling in India

## Overview

KathaVerse is an interactive educational website created as part of the Indian Knowledge System (IKS) project. The website explores the rich tradition of Oral Forms of Storytelling in India, from ancient folk tales and epics to contemporary digital storytelling.

## Features

* Interactive storytelling experience
* Animated landing page
* Smooth scrolling navigation
* Traditional and modern storytelling sections
* Embedded presentation videos
* Background music
* Interactive quiz section
* Responsive design for different screen sizes

## Topics Covered

1. Folk Tales
2. Great Epics
3. Songs and Ballads
4. Theater and Dance
5. Contemporary Oral Storytelling
6. Conclusion and Quiz

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Tailwind CSS

## Purpose

The goal of this project is to present India's storytelling traditions in an engaging digital format while demonstrating how oral storytelling has evolved from traditional community gatherings to modern digital platforms.
